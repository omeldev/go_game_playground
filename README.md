# Game in Go

[Cheatsheet](https://www.raylib.com/cheatsheet/cheatsheet.html)

---

### Features

- Kamera folgt dem Spieler (zentriert auf der aktuellen Sprite).
- Hintergrund wird als gekacheltes, unendliches Scrolling gezeichnet.

### Controls

- `A` / `D`: Laufen
- `W` oder `SPACE`: Springen

### Requirements

```
sudo apt-get install libgl1-mesa-dev libxi-dev libxcursor-dev libxrandr-dev libxinerama-dev libwayland-dev libxkbcommon-dev
```
